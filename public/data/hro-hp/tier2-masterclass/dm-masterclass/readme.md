## Module Structure
1. **DM-101: Decision Fundamentals** - Types, speed vs quality, authority, ownership
2. **DM-102: Decision Models** - OODA, recognition-primed, structured analysis
3. **DM-103: Cognitive Biases** - Confirmation, anchoring, sunk cost, overconfidence
4. **DM-104: Time-Critical Decisions** - Pressure, satisficing, buying time, stress
5. **DM-105: Team Decision Making** - Involvement, dissent, groupthink, commitment
6. **DM-106: Post-Decision Analysis** - Outcomes, debriefs, documentation, improvement
