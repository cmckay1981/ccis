# Well Control Dashboard - System Specification

**Project:** OGLMS Digital Well Control System
**Date:** November 21, 2025
**Status:** Design Specification

---

## Overview

A comprehensive dashboard system where users create and manage well configurations that:
1. **Save to Firebase** - Persistent storage with recall capability
2. **Auto-populate calculators** - Pre-fill well volumes calculator with saved data
3. **Contextualize AI prompts** - Provide well-specific data to Gabriella/CCIS agents
4. **Track operations** - Monitor current well status and activities

### 1. **Auto-Population of Volume Calculator**

When user clicks "Open Volume Calculator" from well workspace:

```javascript
// Frontend code
async function openVolumeCalculator(wellId) {
  // Fetch well data from Firebase
  const wellData = await getWellConfiguration(wellId);

  // Open calculator HTML in new window/tab
  const calcWindow = window.open('well-volumes-calculator.html', '_blank');

  // Wait for calculator to load
  calcWindow.addEventListener('load', () => {
    // Auto-populate all fields
    calcWindow.populateFromWellData({
      units: wellData.units,
      stringComponents: wellData.stringConfig.components,
      wellSections: wellData.wellSections.sections,
      pumpData: wellData.stringConfig.pumpData,
      wellName: wellData.wellName,
      rigName: wellData.rigName
    });

    // Trigger calculations
    calcWindow.recalcAll();
  });
}
```

### 3. **Firebase Integration**

```javascript
// Cloud Function to save complete well configuration
exports.saveCompleteWell = functions.https.onCall(async (data, context) => {
  if (!context.auth) {
    throw new functions.https.HttpsError('unauthenticated', 'Must be logged in');
  }

  const { wellData, stringConfig, wellSections, bopConfig } = data;
  const db = admin.firestore();
  const batch = db.batch();

  // Create well document
  const wellRef = db.collection('wells').doc();
  batch.set(wellRef, {
    ...wellData,
    wellId: wellRef.id,
    createdBy: context.auth.uid,
    createdAt: admin.firestore.FieldValue.serverTimestamp(),
    updatedAt: admin.firestore.FieldValue.serverTimestamp()
  });

  // Create string config
  const stringRef = db.collection('well-string-configs').doc();
  batch.set(stringRef, {
    ...stringConfig,
    configId: stringRef.id,
    wellId: wellRef.id,
    isActive: true,
    createdAt: admin.firestore.FieldValue.serverTimestamp()
  });

  // Create well sections
  const sectionsRef = db.collection('well-sections').doc();
  batch.set(sectionsRef, {
    ...wellSections,
    sectionId: sectionsRef.id,
    wellId: wellRef.id,
    isActive: true,
    createdAt: admin.firestore.FieldValue.serverTimestamp()
  });

  // Create BOP config
  const bopRef = db.collection('bop-configurations').doc();
  batch.set(bopRef, {
    ...bopConfig,
    bopConfigId: bopRef.id,
    wellId: wellRef.id,
    createdAt: admin.firestore.FieldValue.serverTimestamp()
  });

  // Update well with references
  batch.update(wellRef, {
    activeStringConfigId: stringRef.id,
    activeWellSectionsId: sectionsRef.id,
    activeBOPConfigId: bopRef.id
  });

  await batch.commit();

  return {
    success: true,
    wellId: wellRef.id,
    message: 'Well configuration saved successfully'
  };
});
```
